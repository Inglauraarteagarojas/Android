RadioGroup, RadioButton y CheckBox

La guía propone menús de selección. El ejercicio se contextualiza como un configurador de misión para decidir el canal de comunicación y los sensores activos.

1. Selección única y múltiple

RadioGroup + RadioButton

Permiten escoger una sola opción de un conjunto. Ejemplo: Wi‑Fi, LoRa o Bluetooth.

CheckBox

Permite activar varias opciones simultáneas. Ejemplo: GPS, temperatura y humedad.

4. Lógica Java

Fragmento MainActivity.java
RadioGroup grupo = findViewById(R.id.grupoComunicacion);
CheckBox chkGps = findViewById(R.id.chkGps);
CheckBox chkTemperatura = findViewById(R.id.chkTemperatura);
CheckBox chkHumedad = findViewById(R.id.chkHumedad);
TextView salida = findViewById(R.id.txtConfiguracion);
Button guardar = findViewById(R.id.btnConfigurar);

guardar.setOnClickListener(v -> {
    int idSeleccionado = grupo.getCheckedRadioButtonId();
    if (idSeleccionado == -1) {
        salida.setText("Selecciona un canal de comunicación");
        return;
    }

    RadioButton canal = findViewById(idSeleccionado);
    StringBuilder sensores = new StringBuilder();
    if (chkGps.isChecked()) sensores.append("GPS ");
    if (chkTemperatura.isChecked()) sensores.append("Temperatura ");
    if (chkHumedad.isChecked()) sensores.append("Humedad ");

    if (sensores.length() == 0) {
        salida.setText("Selecciona al menos un sensor");
        return;
    }

    salida.setText("Canal: " + canal.getText()
            + "
Sensores: " + sensores);
});

5. Ejercicio orientador

Construye una pantalla denominada Configuración de misión con estas reglas:

Un solo canal

El usuario debe escoger exactamente un medio de comunicación.

Uno o varios sensores

Se pueden activar simultáneamente GPS, BME280/DHT22, presión o batería.

Validación

No se puede guardar si falta el canal o no hay sensores.

Resumen

La app presenta la configuración en un TextView.

Misión PicoSat 4Agrega un CheckBox “Modo ahorro”. Si está activo, el intervalo de envío debe cambiar de 5 a 30 segundos.

6. Autoevaluación

¿Por qué los RadioButton deben estar dentro de un RadioGroup?

Porque el grupo administra la selección única y desmarca automáticamente las demás opciones.

¿Qué componente sirve para seleccionar varios sensores?

CheckBox.

¿Qué ocurre si getCheckedRadioButtonId() retorna -1?

No hay ningún RadioButton seleccionado.
