# Curso Android + PicoSat

Sitio web estático dividido en seis ventanas: Inicio y Unidades 1 a 5.

## Abrir localmente

1. Abra la carpeta en Visual Studio Code.
2. Use la extensión **Live Server** y abra `index.html`.
3. También puede abrir `index.html` directamente, aunque algunos navegadores limitan la vista embebida de PDF cuando se usa `file://`.

## Publicar en GitHub Pages

1. Suba todo el contenido de esta carpeta a un repositorio.
2. En GitHub vaya a **Settings → Pages**.
3. Seleccione **Deploy from a branch**, rama `main` y carpeta `/root`.
4. Espere a que GitHub genere la URL.

## Estructura

- `index.html`: página principal y videos.
- `unidad1.html` a `unidad5.html`: ventanas independientes.
- `styles.css`: diseño visual y loader.
- `script.js`: animación de carga, menú y botones de copia.
- `pdfs/`: guías originales.
- `ejemplos/`: archivos de código para cada unidad.

## Nota de seguridad

No publique contraseñas Wi‑Fi, tokens ni credenciales de Firebase. Ajuste las reglas de seguridad antes de usar la base fuera de un laboratorio.
