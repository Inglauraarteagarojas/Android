EditText, TextView y Button

Esta unidad transforma los componentes básicos de la guía en un simulador móvil de telemetría para el pico satélite.

1. Componentes básicos

EditText

Permite ingresar información. Se usará para temperatura, humedad, GPS o identificador.

Button

Ejecuta una acción cuando el usuario lo pulsa.

TextView

Muestra texto, resultados, estados o alertas.

2. Flujo de interacción

Usuario escribe
EditText
Pulsa Button
Java valida
TextView muestra

La interfaz captura la lectura, el evento del botón activa una función Java y el resultado se presenta en pantalla.

4. Evento y validación en Java

MainActivity.java
public class MainActivity extends AppCompatActivity {
    private EditText edtTemperatura, edtHumedad;
    private TextView txtResultado;

    @Override
    protected void onCreate(Bundle savedInstanceState) {
        super.onCreate(savedInstanceState);
        setContentView(R.layout.activity_main);

        edtTemperatura = findViewById(R.id.edtTemperatura);
        edtHumedad = findViewById(R.id.edtHumedad);
        txtResultado = findViewById(R.id.txtResultado);
        Button btnEvaluar = findViewById(R.id.btnEvaluar);

        btnEvaluar.setOnClickListener(v -> evaluar());
    }

    private void evaluar() {
        String t = edtTemperatura.getText().toString().trim();
        String h = edtHumedad.getText().toString().trim();

        if (t.isEmpty() || h.isEmpty()) {
            txtResultado.setText("Completa los dos campos");
            return;
        }

        double temperatura = Double.parseDouble(t);
        double humedad = Double.parseDouble(h);
        String estado = (temperatura > 45 || humedad > 90)
                ? "ALERTA" : "NORMAL";

        txtResultado.setText("T=" + temperatura + " °C
HR="
                + humedad + "%
Estado=" + estado);
    }
}
ValidaciónNo conviertas texto a número antes de verificar que los campos no estén vacíos.

5. Ejercicio orientador

Construye el formulario

Incluye entradas para temperatura, humedad, latitud y longitud.

Programa el botón

Al pulsarlo, valida los campos y construye un resumen.

Genera una alerta

Si una variable supera el umbral, muestra un mensaje claro.

Prueba casos

Ensaya datos normales, campos vacíos y valores fuera de rango.
