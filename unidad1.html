1. Conceptualización

La Programación Orientada a Objetos permite representar elementos del mundo real mediante clases. Para el proyecto, la clase principal será PicoSatelite.

Clase

Molde que define atributos y comportamientos. Ejemplo: un pico satélite tiene identificador, temperatura, humedad y ubicación.

Objeto

Instancia concreta creada a partir de la clase. Ejemplo: PICO-01.

Constructor

Método especial que inicializa el objeto cuando se usa new.

Método

Acción que el objeto puede realizar, como actualizar telemetría o evaluar una alerta.

Los datos ingresan al objeto, los métodos actualizan sus atributos y una condición decide si el estado es normal o requiere alerta.

PicoSatelite.java
public class PicoSatelite {
    private String id;
    private double temperatura;
    private double humedad;
    private double latitud;
    private double longitud;

    public PicoSatelite(String id) {
        this.id = id;
    }

    public void actualizarTelemetria(double temperatura, double humedad,
                                     double latitud, double longitud) {
        this.temperatura = temperatura;
        this.humedad = humedad;
        this.latitud = latitud;
        this.longitud = longitud;
    }

    public String evaluarEstado() {
        if (temperatura > 45) return "ALERTA: temperatura alta";
        if (humedad > 90) return "ALERTA: humedad alta";
        return "Sistema estable";
    }

    public String resumen() {
        return id + " | T=" + temperatura + " °C | HR=" + humedad
                + "% | GPS=" + latitud + "," + longitud;
    }
}
Main.java
public class Main {
    public static void main(String[] args) {
        PicoSatelite sat = new PicoSatelite("PICO-01");
        double[] temperaturas = {22.5, 24.1, 46.2};

        for (int i = 0; i < temperaturas.length; i++) {
            sat.actualizarTelemetria(temperaturas[i], 61.0, 4.7109, -74.0721);
            System.out.println(sat.resumen());
            System.out.println(sat.evaluarEstado());
        }
    }
}

5. Autoevaluación

¿Cuál es la diferencia entre clase y objeto?

La clase es el molde; el objeto es una instancia creada a partir de ese molde.

¿Para qué sirve un constructor?

Inicializa un objeto cuando se crea y permite asignar valores iniciales.

¿Cuándo usarías un ciclo?

Cuando necesitas repetir un proceso, por ejemplo analizar una secuencia de lecturas.
